Idea Management System

Innovation Management for Organizations

A Complete Web-Based Solution for Managing Business Ideas

SEM-6 | Innovation Management Subject

Project Overview

The Idea Management System is a web-based application designed to help organizations collect, evaluate, and manage innovation ideas through a structured innovation funnel.

Purpose

  • Enable employees to submit and share innovative ideas
  • Provide structured evaluation of ideas (Impact & Feasibility)
  • Track ideas through innovation stages
  • Foster collaborative feedback via comments and voting
  • Generate insights from innovation metrics

Key Features - Part 1

1. Idea Submission

Submit ideas with:

  • Title & Description
  • Category (Product, Process, Marketing, Social)
  • Impact & Feasibility scores (1-5)
  • Initial rating and funnel stage

2. Idea Screening & Scoring

Automatic Screening Score calculation:

  • Score = Impact + Feasibility (Max: 10 points)
  • Helps identify most promising ideas
  • Data-driven decision making

Key Features - Part 2

3. Innovation Funnel Tracking

4-Stage Pipeline:

  • Idea → Initial submission
  • Screening → Evaluation phase
  • Prototype → Development stage
  • Implementation → Execution phase

4. Community Engagement

  • Add comments to any idea
  • Upvote/Downvote ideas
  • Collaborative feedback system
  • Build organizational support

Key Features - Part 3

5. Search & Filter

  • Search by title or description
  • Filter by category
  • Filter by funnel stage
  • Combined filtering for precision

6. Sorting & Analytics

  • Sort by: Newest, Rating, Score, Votes
  • Ascending/Descending toggle
  • Top 5 Ideas page (dashboard view)
  • Real-time statistics & funnel visualization

7. Data Management

  • Edit any idea anytime
  • Delete ideas
  • Export all ideas as JSON report
  • Reset data to defaults

Dashboard & Analytics

Innovation Overview Panel

Real-time metrics tracking:

📊 Total Ideas

Count of all submitted ideas

⭐ Average Rating

Mean rating across all ideas

📈 Avg Screening Score

Impact + Feasibility average

🗳️ Total Votes

Community engagement metric

Innovation Funnel Visualization

Visual bar charts showing ideas distribution across each stage with percentage breakdown
