Help & Keyboard Shortcuts

✨ Features

  • Add Ideas: Fill out the form and click "Add Idea"
  • Edit Ideas: Click "Edit" button on any idea card
  • Delete Ideas: Click "Delete" button (with confirmation)
  • Vote: Use +/- buttons to upvote or downvote
  • Comment: Add comments at the bottom of each idea card
  • Search: Type in the search box to filter ideas
  • Filter: Use category and stage dropdowns
  • Sort: Choose sort criteria and direction
  • Export: Download all data as JSON file
  • Reset: Restore sample data

⌨️ Keyboard Shortcuts

  • Escape: Close any open modal
  • Ctrl/Cmd + K: Focus on search box

📊 Scoring System

Screening Score = Impact + Feasibility (Max: 10 points)
Higher scores indicate more promising ideas for implementation.
